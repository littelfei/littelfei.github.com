数据类型 转换为true的 转换为false的
String 任何非空字符串 “”（空字符串）
Number 任何非零数值（包括无穷大） 0和NaN
Object 任何对象 null
undefined – undefined

这是一个闭包,闭包可以用在许多地方.它的最大用处有两个,一个是可以读取函数内部的变量,另一个就是让这些变量的值始终保持在内存中.
